# React-Redux-TodoApp


### Tech Stack

1. React & Redux
2. Node.js & Express
3. MongoDB & Mongoose
4. Webpack (bundling)
5. BootStrap for Styling

### Instructions to setup
##### Prerequisites
1. Node version > 4
2. mongodb installed

##### Steps
1. `npm install`
2. `mongod` (to start up mongodb instance)
3. `npm run start`
4. Goto `http://localhost:3000`
